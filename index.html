Téléchargement

Télécharger G19 Neo Dashboard

Le logiciel se lance sur le PC. Le téléphone se connecte ensuite à l'adresse locale affichée par l'application ou via le QR code généré par le PC.

  1. Télécharger le ZIP du logiciel.
  2. Extraire le dossier sur le PC Windows.
  3. Lancer l'installation des dépendances si nécessaire.
  4. Démarrer l'application puis scanner le QR code avec le téléphone.

À propos

Un projet local, simple et évolutif

G19 Neo Dashboard est né de l'idée de recycler un smartphone comme écran secondaire, dans l'esprit des anciens claviers Logitech G19, mais avec une interface web moderne.

Le projet privilégie la simplicité : un PC qui héberge le tableau de bord, un téléphone connecté au même réseau, et des fonctions pratiques sans compte externe.
